# Cеминар 8 по SysDеsign

## Описание проекта
Работа со спутниками, их группировками и выполнением миссий.
Проект содержит два подпроекта:
- space-api (основной сервис), запускается на порту 8080;
- space-scheduler (планировщик), запускается на порту 8081.

## Дополнительно реализовано:
- добавлена валидация конфигурации: проверка, что для SINGLE_SATELLITE указан satelliteName, а для CONSTELLATION – нет;
- добавлены тесты для контроллера в space_api.
